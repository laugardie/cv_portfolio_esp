Portfolio

Para diseñar mi portfolio comencé diseñando mi marca personal para crear un estilo que me representara y estuviera en consonancia con mi personalidad.

Brand Personality

El primer ejercicio que utilicé para diseñar mi portfolio fue un brand personality utilizando The five dimensions of brand personality de Jennifer Aaker. El ejercicio trata de elegir 2 adjetivos que representen tu marca y 1 adjetivo que no la represente. En este caso:

Soy cautivadora y organizada pero nunca inflexible.

Brand Affinity

El segundo ejercicio que realicé fue un brand affinity. Con este ejercicio podemos definir la personalidad de una marca haciendo referencia a personas, cosas e incluso otras marcas que percibamos que tienen una personalidad similar. Así que, si fuera una persona famosa sería Mónica Geller de Friends, si fuera un animal sería un gato de bengala y si fuera otra marca sería Muji.

Moodboard

Una vez claro mis atributos y mi brand affinity, creé un moodboard con imágenes donde se vieran representadas estas características.

Para confirmar que el moodboard y los atributos de mi marca estaban en sintonía realicé un desirability test con mis compañeros de Ironhack. Algunos de los adjetivos que utilizaron para describir este moodboard son:organizado, orden, nordic, acogedor, hygge y relajado.

Style tile

A través de las imágenes del moodboard saqué la paleta de colores de mi marca personal creando así el style tile. Como tipografía elegí Open Sans una Sans serif con formas abiertas y neutrales con una apariencia amigable y gran legibilidad.

Pantallas de error

Para inspirarme sobre el mood que quería que mi portfolio desprendiera, diseñé dos pantallas de error en Sketch.

Logo

Para diseñar mi logo utilicé illustrator para trazar una fotografía personal. Aquí podéis ver cómo tracé el contorno de la imagen y luego rellené la imagen dándole ese toque de color.

Diseño del portfolio

Por último, antes de empezar a programar el portfolio. Diseñé la estructura utilizando Sketch. Una vez terminado, lo exporté a Zeplin para poder trabajar de una manera más cómoda a la hora de programar la página.

HTML y CSS

En la séptima semana del Bootcamp de UX/UI de Ironhack, estuvimos aprendiendo a programar en HTML y CSS. Durante esta semana programamos nuestro portfolio personal utilizando herramientas como: Zeplin, iterm, Visual Studio y Github. Fue una semana muy intensa pero aprendimos un montón de cosas. Además fue muy satisfactorio ver el resultado final. Aquí os dejo el Github del proyecto.
